# 1984

Программное средство представляет собой клиент–серверную систему для мониторинга пользовательской активности в локальной корпоративной сети, обеспечивающую сбор и передачу в реальном времени данных о состоянии рабочих станций (IP, пользователь, домен, хост, время последней активности), а также возможность удалённого получения скриншотов; клиентская часть, реализованная на C++ с использованием WinAPI и WinSock, фиксирует изменения активности пользователя и отправляет их на сервер, тогда как серверное приложение на C# (.NET, WPF) выполняет приём, отображение и хранение информации, включая запись событий активности (last_active_time) с привязкой к рабочим станциям в базе данных, что позволяет централизованно отслеживать состояние подключённых клиентов и анализировать их поведение
